## Custom environment configuration

This container support setting certain custom environment variables with the use of [drone/envsubst](https://github.com/drone/envsubst).

| Name                            | Default  |
|---------------------------------|----------|
| SONARR__ANALYTICS_ENABLED       | `False`  |

| SONARR__BRANCH                  | `main`   |

| SONARR__PORT                    | `8989`   |
